About

This webapp was developed using Python and Django framework with Bootstrap. It utilises the Cobalt API.

Cobalt API Docs
Notes

Since this utilizes an API, the results may load slower depending on how many requests are made on the API and the amount of "layers" the course comes with. For example: a fourth year class with a lot of prerequisites will load slower than a fourth year course with only one prerequisite. If there are enough people using this, I will try to learn on how to self-host it or create a database to make things run faster

If there are courses missing or fail to load, the API may not have that information, the course no longer exists or I just need to improve my code.

Current Issues

Currently, some 4th year level courses especially Rotman courses aren't responding very well therefore some prerequisites fail to load. In addition CSC423H5 is currently displaying incorrect prerequisites due to the API returning the initial prerequisites without the "CSC". Therefore the program searches and grabs the wrong course code

This tool so far is now limited to 4 "layers". The more layers there are the slower the site loads.
